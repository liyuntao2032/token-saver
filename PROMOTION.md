**V1.0思路：** 保留完整句子
```python

    保留整个句子
```

**V1.1.0思路：** 只提取关键字段
```python
decision = re.search(r'(?:决定|确认)[：:\s]*([^\n，。]{3,20})', text)
data['decision'] = decision.group(1)
```
